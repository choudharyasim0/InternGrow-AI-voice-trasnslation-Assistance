# 🌍 AI Voice & Text Translation Assistant

A modern multilingual translation web application built with **Python Flask** that allows users to translate text, use voice input, listen to translated speech, and manage translation history through a clean and responsive interface.

---

## 📌 Project Overview

AI Voice & Text Translation Assistant is designed to make communication easier across different languages. It combines text translation, browser-based voice recognition, and text-to-speech into a single easy-to-use web application.

The project was developed as part of the **InternGrow Internship Program** to demonstrate practical web development and Python skills.

---

# ✨ Features

- 🌍 Translate text into multiple languages
- 🎤 Voice Input (Speech Recognition)
- 🔊 Text to Speech (Listen)
- 📜 Translation History
- 🌙 Dark / Light Mode
- 📱 Responsive UI
- 📋 Copy Translation

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Voice-Text-Translation-Assistant.git
```

### Move to Project

```bash
cd AI-Voice-Text-Translation-Assistant
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate

Windows

```bash
.venv\Scripts\activate
```

Linux / Mac

```bash
source .venv/bin/activate
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python app.py
```
